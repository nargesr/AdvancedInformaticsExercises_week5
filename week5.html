Plotting basics in base R

plot(sin)

Change it so that the x axis label is \(\sigma\) and the y axis label is \(\tau\). Those are the Greek letters sigma and tau, respectively.

plot(sin, xlab = expression(sigma), ylab = expression(tau))

Take the same plot, and change the x axis label to \(\sigma^2_s\).

plot(sin, xlab=expression(sigma[s]^2))

Color and line style (base graphics)

For the same plot as above, change the plot so that the line is red, dot-dash, and 3 times the default line thickness.

plot(sin, col = "red", lty = 4, lwd = 3)

Axis labels from data

Take the graph we started with, and assume theres a variable y = 7. Add a title to the graph that says \(\Gamma = 7\). (Thats the Greek letter gamma.) Do it using the variable! Dont hard-code in the value!

y = 7
plot(sin, main = bquote(Gamma == .(y)))

Annotations from data

Take the graph we started with, and assume theres a variable y = 7. Add a text element at coordinates x=0.1, y=0.4 to the graph that says \(\Gamma = 7\).

y = 7
plot(sin)
text(x=0.1, y=0.4, bquote(Gamma == .(y)))

R/ggplot

Take the following ggplot:

library(ggplot2)
data(mtcars)
p = ggplot(mtcars, aes(x=cyl, y = mpg)) + geom_point()
print(p)

Repeat the various manipulations above. Where the above instructions refer to lines, change point color/size/shape instead.

p1 = p + xlab(expression(sigma)) + ylab(expression(tau))
print(p1)

p2 = p + xlab(expression(sigma[s]^2))
print(p2)

p3 = p + geom_point(color='blue', size=2, shape= 15)
print(p3)

y = 7
p4 = p + ggtitle(bquote(Gamma == .(y)))
print(p4)

y = 7
p + annotate("text",label = bquote(Gamma == .(y)), x = 4.5, y = 15)
